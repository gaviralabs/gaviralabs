Security

Última actualización: 2026-02-20

Last updated: 2026-02-20

La seguridad es una prioridad. Si detectas una vulnerabilidad, gracias por reportarla de forma responsable.

Canal de reporte

  • Email: gaviralabs@gmail.com
  • Idioma: ES / EN

Qué incluir

  • Descripción clara del impacto y alcance.
  • Pasos para reproducir (PoC si es posible).
  • Versión/navegador/dispositivo (si aplica).

Buenas prácticas

  • No accedas a datos de terceros ni interrumpas el servicio.
  • No publiques la vulnerabilidad antes de que podamos investigarla.
  • Intentaremos responder lo antes posible y coordinar una corrección.

También publicamos un fichero estándar en /.well-known/security.txt.

Security matters. If you find a vulnerability, please report it responsibly.

Reporting

  • Email: gaviralabs@gmail.com
  • Languages: ES / EN

Include

  • Clear description of impact and scope.
  • Reproduction steps (PoC if possible).
  • Version/browser/device (if relevant).

Rules

  • Do not access other users’ data or disrupt the service.
  • Please do not disclose publicly before we investigate.

A standard file is also available at /.well-known/security.txt.
